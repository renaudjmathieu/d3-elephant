#  Animated SVG with D3 + React

An animated SVG drawn with D3 + React and CSS.

![image](https://raw.githubusercontent.com/renaudjmathieu/my-web-app/main/www/src/img/d3-elephant.png)

## :technologist: Available Script Commands

In the project directory, you can run:

### `npm install`

Install the necessary packages by following these steps:

In the Terminal, go to the project folder:
```	
cd d3-elephant
```	
Install the necessary packages using `package-json`:
```	
npm install
```	

### `npm run dev`

Runs the app in development mode by following these steps:

In the Terminal, go to the project folder:
```	
cd d3-elephant
```	
Run the app in the development mode: 
```	
npm run dev
```	
*Open [http://localhost:8080](http://localhost:8080) to view it in the browser.\
The page will reload when there are changes.\
You may also see any lint errors in the console.*

### `npm run build`

Builds the app for production to the `dist` folder by following these steps:

In the Terminal, go to the project folder:
```	
cd d3-elephant
```	
Run the app in the development mode: 
```	
npm run build
```
